Pipeline Team Notes

📋 Adding / Removing Agents

When a manager adds or removes an agent from their team, notify the Pipeline team immediately with:

• Agent name
• Start date (or end date)
• Manager / Team Lead they're under

Keeps routing clean and organized.

🔄 Split Management

Some agents are under split management (e.g., Brandon Ogletree / Colton Ramon, Dalston Wells / Brooks Pedersen). Jordan Andersen serves as recruiter across several groups. Confirm with both parties when routing or escalating calls.

🚨 Escalation Path

Agent issue → Team Lead → Manager → Pipeline Team

When escalating a call always include:
• Caller ID (phone number)
• Short note on the issue
• Recording if available

Response expectation: 1–2 hours.
